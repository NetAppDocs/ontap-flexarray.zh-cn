= ONTAP支持的阵列LUN类型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您只能将存储阵列LUN映射到ONTAP。如果LUN 0是存储类型的LUN、则可以将其映射到ONTAP。

某些存储阵列具有非storage _command_ LUN。您不能将命令类型LUN映射到ONTAP系统。
