= 在主机之间共享存储阵列
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
典型的存储阵列可为不同供应商的主机提供存储。但是、ONTAP要求某些存储阵列专用于ONTAP系统。

要确定供应商的存储阵列是否必须专用于ONTAP系统、请参见_互操作性表_。
