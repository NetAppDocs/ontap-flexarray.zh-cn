= 规划使用阵列LUN的配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要规划使用阵列LUN的ONTAP配置、您必须验证特定ONTAP版本是否支持存储阵列。此外、您还必须检查互操作性和支持信息、以确保所有系统都满足使用阵列LUN的配置的要求。
