= 具有两个双端口阵列LUN组的独立系统
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在采用简单光纤连接配置的独立ONTAP系统中、ONTAP系统上的每个FC启动程序端口对都会访问一个单独的阵列LUN组。

此配置支持与互操作性表中列出的所有存储阵列一起使用、就像在系统上运行的ONTAP版本支持的那样。
